# AI Code Agent with Tool Calling and Planning Loop (Python, Google Gemini)

A Gemini-powered autonomous CLI coding agent that analyzes tasks, selects tools, and iteratively modifies project files.

---

## Overview

This project implements a multi-step reasoning agent inspired by Claude Code. The agent uses Google Gemini to interpret tasks, plan actions, call tools, and modify codebases autonomously.

---

## Features

- Tool-calling architecture  
- Multi-step planning loop  
- File read/write support  
- LLM-driven decision making  
- Python CLI interface  

---

## Tech Stack

- Python  
- Google Gemini API  
- uv project manager  
- Linux (WSL2)  
- VS Code Remote  
- Boot.dev CLI
